# Projeto: background
## Descrição
  - Constroi fila para realização de funcionalidades em background, como envio de email ou qualquer outro coisa que queira
  fazer enquando seu servidor é executado.
## Tecnologias útlizadas:
  - nodejs
  - docker
  - postgres
  - redis
  - bull
  - bull-board
  - dotenv
  - express
  - nodemailer
  - nodemon
  - npm-run-all
  - sucrase
